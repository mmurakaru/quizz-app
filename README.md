This project is about building a Quiz-App.

The App includes 4 pages: 

Home > index.html
Quiz creator page > create.html
Profile settings > profile.html
Bookmarked Questions > bookmarks.html

The objective of the app to include questions und show the answer on action. 
The user can create new quiz-questions and there related answers and categories the questions/answers with #tags. In case a questions has to be safed for later the user can bookmark it on a separate page.
